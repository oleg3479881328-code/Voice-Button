## Detailed Description

Voice Button adds a small dictation control next to an editable text field on an ordinary web page.

Click into a text field, use the microphone control, speak, and the recognized text is inserted into the field.

Current functionality includes:

- a floating voice button near an active editable field;
- manual and automatic language controls;
- local saving of launcher and language preferences;
- support for ordinary text fields and common editable page areas.

Voice Button is intended for simple voice input inside Chrome web pages. It does not provide transcription for native desktop applications.
