# 08 — Knowledge Extract

## Local Lesson

A small browser extension becomes difficult to finish when behavior fixes, tester delivery and store preparation live only in chat or issue threads.

## Reusable Pattern Candidate

Before store submission, keep a short evidence-based package:

- actual product purpose;
- current behavior requiring validation;
- tester installation path;
- store-facing documents and missing assets;
- one explicit release gate.
